## 概要
　Cやアセンブラで組込系ソフトウェアやモニタリングソフトの開発経験を経て、私立大学の情報インフラ部門に入職し設計・運用に従事。オンプレミスにてゼロからのネットワーク・サーバ構築や管理ツールの開発、学内の教職員・授業用コンピュータ・ネットワークシステムの導入・リプレイス業務や、大学事務局の要望に応じた部署用ウェブシステムの開発など技術面のみならず、キャンパス所在地である東京都青梅市民に向けたIT講習会や、文系・芸術系学生への情報リテラシー教育なども長期に渡り経験。
　その後ゲーム開発会社にてブラウザゲームのデータセンターでのインフラ構築、Amazon EC2でのインフラ構築、運用管理、CSツールやAPI開発に従事。数度の転機によりウェブ（フロント・バックエンド）開発業務に従事し、現在に至る。

## その他
### 自己PR
　小学5年の頃から親しんできたコンピュータ技術、その後20代前半より触れることになったインターネット、UNIX、そしてインターネット・ネットワーク技術、そしてそれらに付随する技術は当時より常に追いかけ、いつでも現場で活用できるよう心がけています。
　たとえば、今では当たり前のインフラとなったクラウドコンピューティングですが、開発業務と運用業務の効率化として誕生したDevOpsの概念なくしては、開発からリリース、そしてそのインフラ運用管理という一体化システムの迅速かつ正確な継続が困難となりつつあります。こうした概念から誕生した新たな仮想化技術やコンテナ技術などを現場に応用できるよう、日々情報収集と個人的探求に務めております。
　また、コンピュータ言語は昔からの興味対象であり、構文解析などは専門学校の卒業論文のテーマでもありました。新たな言語が登場すれば極力触れるように心がけています。その中で興味が湧いた言語については、さらに追求しています。
　また、ソフトウェア開発手法や合理化などの概念・技術や、可能ならコンピュータサイエンス・ソフトウェアエンジニアリングなど、興味のままに知識の幅を広げております。エンジニアによっては、こうした取り組みが勉強や努力することの一環と捉えている方もおられるかもしれませんが、私にとってはITに関する新たな知見の獲得は常に普通のことであり、生活の一部になっています。
